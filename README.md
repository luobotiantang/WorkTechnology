    三人行，必有我师”，出自于《论语·述而》。原文是：“子曰：三人行，必有我师焉；择其善者而从之，其不善者而改之。”意思是：孔子说：
    “别人的言行举止，必定有值得我学习的地方。选择别人好的学习，看到别人缺点，反省自身有没有同样的缺点，如果有，加以改正。
